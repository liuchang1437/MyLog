#工作&生活 记录
自*2016年4月26日*

## 简介

本项目以周为单位，分为两个模块——Work & Life，记录我的日常工作和生活状态。

## 格式

1. 文档**名称**规则：

   `<start_month_day>-<end_month_day>.md`

   例如：`0425-0429.md`

2. 文档**内容**规则：

   1. Header:

      ```
      ########## 起止日期 ##########

      eg:
      ########## 04.25 - 05.01 ##########
      ```

   2. Body:

      ```
      周#（日期）：
      1. 时间  记录1;
      2. 时间  记录2;
      3. ...
      ```
